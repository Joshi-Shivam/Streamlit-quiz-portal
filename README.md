🚀 Interactive Multi-Category Quiz App
A dynamic, fully interactive web application built with Python and Streamlit that transforms traditional trivia into an engaging, gamified user experience. The application features multiple academic and entertainment categories, optimized front-end asset delivery, and an immersive retro arcade audio aesthetic.

🛠️ Tech Stack & Architecture
Frontend & Framework: Streamlit (Python)

State Management: Localized procedural execution logic to eliminate state-rerun math compounding bugs.

Multimedia: Lightweight bit-rate compressed audio assets paired with high-resolution contextual imagery.

✨ Key Features
Dual-Column Grid Layout: Re-engineered traditional linear form layouts into a responsive, side-by-side view (Questions on the left, progress bars and visual assets on the right) to significantly reduce vertical scrolling and maximize user engagement.

Dynamic Performance Matrix: Utilizing custom Streamlit components (st.form, st.dialog, and st.metric), the app evaluates submissions and instantly fires a real-time, color-coded delta metrics grid charting the user’s question-by-question performance alongside animations.

On-Demand Cheat Sheet: Built an interactive, collapsible post-submission dropdown menu using st.expander allowing users to cleanly audit correct answers without cluttering the main screen.

Clear User Redirection: Implemented structural post-game navigation split into informational columns, ensuring clear user call-to-actions (resetting to clear scores or switching categories via the sidebar) to completely prevent user experience dead-ends.
